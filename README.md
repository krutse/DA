# Портфолио аналитика данных

### 1. [Выгрузка данных с сайта по API в PostgreSQL](https://github.com/krutse/DA/tree/main/%D0%92%D1%8B%D0%B3%D1%80%D1%83%D0%B7%D0%BA%D0%B0%20%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85%20%D0%BF%D0%BE%20API)

Проект, демонстрирующий навыки владения Python, SQL, Google Sheets API, smtplib.

Задача: Создание скрипта сбора данных с сайта по API и загрузка в БД для дальнейшей обработки, формирование лога обработки данных. 
По результатам сбора данных сформировать аггрегированные данные в google sheets и отправить файл по е-mail. 

### 2. [Автоматизация обработки чеков торговой сети](https://github.com/krutse/a_d_ret_cn)

Проект, демонстрирующий навыки владения Python, SQL, планировщиком заданий cron.

Задача, автоматизировать обработку данных по продажам, обработанные данные загрузить в базу данных.

### 3. [PowerBI dashboard (Дашборд для руководителей и маркетологов для оценки розничных продаж)](https://github.com/krutse/DA/tree/main/PowerBI%20dashboard)

Проект, демонстрирующий навыки предобработки данных и построения дашбордов в Power BI.

Дашборд предоставляет ключевые мерики по продажам:
- показывает динамику продаж/прибыли, в том числе по магазинам;
- ABC/XYZ анализ по товарным позициям;
- Динамику заказов и среднего чека.
- Предоставляет актуальную и визуально понятную картину продаж.

### 4. [AW BI "Аналитика продаж ‭фармацевтической компании"»](https://aw-bi.ru/awbi-astrazeneca/d5)       

Проект в рамках конкурса-хакатона «ФАРМАДАШ», демонстрирует навыки построения дашбордов в AW-BI.
