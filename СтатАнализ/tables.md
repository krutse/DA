Описание таблиц БД apteka

Таблица: bonuscheques
|     Поле     |               Описание                       |
|--------------|----------------------------------------------|
|   datetime   | дата и время совершения транзакции           |
|    shop      | название аптеки                              |
|    card      | номер бонусной карты                         |
|bonus_earned  | количество начисленных бонусов               |
|bonus_spent   | количество потраченных бонусов               |
|    summ      | сумма чека                                   |
|summ_with_disc| сумма чека с учетом скидок и списаний бонусов|
|    doc_id    | номер документа                              |

Таблица:employee
|   Поле  |Описание                    |
|---------|----------------------------|
|emp_tabn | табельный номер сотрудника |
| emp_fio | ФИО сотрудника             |

Таблица:shops
| Поле| Описание             |
|-----|----------------------|
| id  | идентификатор аптеки |
|name | название аптеки      |

| DR_Suppl  | поставщик                                    |
| DR_Prod   | производитель                                |
| DR_Kol    | кол-во проданного товара                     |
| DR_CZak   | закупочная цена                              |
| DR_CRoz   | розничная цена                               |
| DR_SDisc  | сумма скидки                                 |

| DR_BCDisc | штрихкод скидки                              |
