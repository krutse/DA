
# Описание задачи
 Дашборд для руководителей и маркетологов, позволяет оценить эффективность розничных продаж.
 Предоставляет ключевые мерики по продажам: 
- показывает динамику продаж/прибыли, в том числе по магазинам;
- ABC/XYZ анализ по товарным позициям;
- Динамику заказов и среднего чека.

Предоставляет актуальную и визуально понятную картину продаж.

## Первая страница - основные показатели
### Показатели
1. Выручка отчётного года в сравнении с прошлым годом
2. Прибыль отчётного года в сравнении с прошлым годом
   - *Позволяют понять насколько выручка/прибыль выросла или упала в долгосрочной перспективе, исключая влияние краткосрочных колебаний или сезонности*
3. Маржа отчётного года в %
   - *Для оценки операционной эффективности*
4. Количество заказов и средний чек отчётного года
   - *Оцениваем активность бизнеса и эффективность продаж.*
5. График - динамика выручки/прибыли 
6. Выручка по магазинам 

### Фильтры
1. Выбор отчётного года
2. Выбор магазина


## Вторая страница - ABC/XYZ анализ 
### Показатели
1. ABC/XYZ анализ по товарным позициям.
   - *Показывает баланс между финансовой значимостью товара (ABC) и операционной эффективностью его управления/прогнозирования (XYZ) для максимизации прибыли и *минимизации издержек бизнеса.**
 *Для ABC анализа берутся продажи за год. Для XYZ анализа анализируются помесячные продажи в течение года.*
 *Значение коэффициента вариации для группы X выбрано <= 0.3, для Y - <= 0.5*	
2. Количество различных товаров в товарных группах
3. Топ 5 лидеров продаж по выручке
4. Топ 5 аутсайдеров продаж по выручке

### Фильтры
1. Выбор отчётного года
2. Выбор магазина
3. Выбор группы товара

## Третья сраница  - динамика среднего чека
### Показатели
1. Таблица - динамика среднего чека
2. График динамика среднего чека
3. График динамика количества заказов

### Фильтры
1. Выбор глубины сравнения параметра (2 последних года)
2. Выбор магазина
